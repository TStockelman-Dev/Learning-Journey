**Title: User Configuration Manager**
The User Configuration Manager is one of the five certification projects that I completed as part of freeCodeCamp's Python Certification Course. This project was largely an exercise in the use of dictionaries and functions. This project was significant as an introduction to the way that Python can gather and process data. The Python logic presented here can be applied to the settings of multiple devices.

**Uses**
This project contains the logic that allows the user to add, remove, or change settings on a device.
