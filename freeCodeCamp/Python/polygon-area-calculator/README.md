**Title: Polygon Area Calculator**

This Polygon Area Calculator is the second of the five certification projects that I completed for the freeCodeCamp Python Certification. This project uses Object-Oriented Programming (OOP), and specifically the concept of inheritance, to calculate the number of times that a square with sides of a given size will fit into a rectangle of a given width and height. Its logic can be expanded to be useful in multiple areas.
