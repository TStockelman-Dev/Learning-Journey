**Title: Universe Database**

This is the first of five certification projects that I completed towards freeCodeCamp's Relational Databases certification. This database was created using PostgreSQL and contains interesting and fun information about the galaxies, stars, planets, moons, and comets that make up our universe, with each of these occupying a separate table within the database. The tables include information concerning the name, size, and distance from earth for the vast majority of the bodies listed. Also included are over 40 "fun facts" about the various galaxies, stars, planets, moons, and comets that make up our universe. This project, overall, was an excellent introduction to PostgreSQL and provided an opportunity to put my understanding of Primary and Foreign keys, normalization, constraints, and other important SQL concepts.
